Exception Hierarchy

All exceptions inherit from the base FrankaException:

Text Representation

Exception (Python built-in)
└── FrankaException
    ├── CommandException
    ├── ControlException
    ├── NetworkException
    ├── InvalidOperationException
    └── RealtimeException
